# Backend-Loom

This repo contains the **Integration Pipeline** project (Next.js + Postgres + worker) used for a Loom walkthrough.

## Project

- **App folder**: `integration-pipeline/`
- **What it does**: signed webhook ingestion → durable event storage → async job processing (locking + retries) → dashboard visibility
